# Game:Ocean-eys

Мы разработали игру-рогалик под названием Ocean eys. Цель игры - пройти два уровня и дойти до босса.

## **Установка и запуск**

Для установки игры необходимо скачать зип файл -> разорхивируйте его -> Запустите файл Ocean.exe

## **Управление**

`w` - Прыжок.

`a` - Ходить влево.

`d` - Ходить вправо.

`ЛКМ` - Бить врага мечем.

`ПКМ` - Выстрелить из пистолета.
